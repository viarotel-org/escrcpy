在 macOS 上使用 scrcpy ​

安装 ​

从官方发布版本安装 ​

下载最新版本的静态构建包：

  • scrcpy-macos-aarch64-v3.2.tar.gz (aarch64 架构)
    SHA-256: f6d1f3c5f74d4d46f5080baa5b56b69f5edbf698d47e0cf4e2a1fd5058f9507b

  • scrcpy-macos-x86_64-v3.2.tar.gz (x86_64 架构)
    SHA-256: e337d5cf0ba4e1281699c338ce5f104aee96eb7b2893dc851399b6643eb4044e

下载完成后解压即可。

注意：macOS 版的 scrcpy 静态构建包仍处于实验阶段。

通过包管理器安装 ​

Scrcpy 可通过 Homebrew 安装：

bash
brew install scrcpy

你需要确保 adb 工具已在 PATH 环境变量中。如果尚未安装：

bash
brew install --cask android-platform-tools

此外，Scrcpy 也可通过 MacPorts 安装，该工具会自动配置 adb：

bash
sudo port install scrcpy

如需手动构建和安装，请参阅 build。

运行 ​

请确保你的设备满足使用前提。

安装完成后，在终端中运行：

bash
scrcpy

或带参数运行（以下示例为禁用音频并录制到 file.mkv）：

bash
scrcpy --no-audio --record=file.mkv

命令行参数的文档可通过以下方式查看：

  • man scrcpy
  • scrcpy --help
  • GitHub 上的文档
